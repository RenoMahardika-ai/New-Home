# Undangan Ulang Tahun Anak (1 Tahun) + Perkenalan Rumah Baru

## Isi yang perlu Anda ganti
1. **Tanggal & Jam acara**
   - Buka `index.html`
   - Ubah bagian berikut:
     - `id="eventDate"` → tanggal acara
     - `id="eventTime"` → jam acara

2. **Foto galeri**
   - Ganti file di folder `assets/` dengan nama:
     - `gallery-1.jpg`
     - `gallery-2.jpg`
     - `gallery-3.jpg`
     - `gallery-4.jpg`

3. **Lokasi**
   - Sudah dipasang menggunakan tautan:
     - https://goo.gl/maps/EkL4Rf1SxeyqMmPs5

## Menjalankan
- Paling mudah: **double-click** `index.html` lalu buka di browser HP.
- Atau pakai server sederhana (opsional) jika diperlukan:
  - dari folder `birthday-invitation-1th`, jalankan:
    - `npx serve .`

## Catatan
- Website ini responsive (mobile-friendly) dan galeri bisa digeser/next-prev.
